# python.RentCalculator 
A CLI-based Rent Calculator built using Python that helps split rent among multiple users by performing basic arithmetic calculations.
## Feature:
* Calculates rent per person
* Handles user input efficiently
* Simple and beginner-friendly CLI application

## Tech Stack:
* Python
* Command Line Interface (CLI)

## How it works?
The user inputs:
* Total rent
* Number of people
